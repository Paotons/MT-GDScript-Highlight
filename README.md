# GDScript 语法高亮 for MT管理器

一份为 MT管理器 定制的 GDScript 语法高亮配置。

## 特性

- 支持 Godot 4.x GDScript 语法
- 区分 GDClass / 自定义类 / 基础类型 三种颜色
- 节点路径 `$Node` 和 `$"Node"` 双格式支持
- 字符串内转义序列单独高亮
- 数字高亮（支持下划线分隔、科学计数法）
- 类型提示 `: type` 和 `-> type` 正确识别
- 信号 `signal` 定义高亮

## 安装

1. 复制 `gdscript.json` 到 MT管理器 的语法配置目录
2. MT管理器 → 编辑器设置 → 语法高亮 → 选择 GDScript

## 已知限制

- MT管理器 的正则引擎不支持复杂负向断言，部分边界匹配可能不完美
- 嵌套注释无法高亮（引擎限制）
